Ejercicio 1

Crea un programa en Java para trabajar con ficheros de acceso aleatorio,
RandomAccessFile, que cambie las palabras del fichero original “ejer1.txt”
por su equivalente en mayúsculas.
Para ello se pide por teclado una palabra, la busca en el fichero de texto
“ejer1.txt” y si existe la modifica escribiéndola en mayúsculas cada vez que
aparece en el fichero.

Ayuda -> Para hacer el cambio de la palabra por su equivalente en mayúsculas, el
programa lee el fichero por líneas. Para cada línea leída se comprueba si contiene la
palabra buscada y si es así se modifica y se sobrescribe la línea completa modificada.
Revisa los métodos de la clase StringBuilder que te pueden resultar útiles.
Una vez realizado el proceso deberás mostrar al usuario que ha funcionado la
transformación.
